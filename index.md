# 📊 cluse - Track your Claude token usage simply

## 📋 About This App

Cluse provides a clear view of your Claude Code token usage. Many users find it difficult to track their spending and activity within the terminal. This tool displays your costs and usage in real-time. It combines a desktop application with a terminal interface to give you full control over your development environment.

## 💻 System Requirements

Cluse runs on Windows 10 or 11. You need a stable internet connection to track your data. The application uses a small portion of your system memory to provide updates without slowing down your computer. No special hardware or professional graphics cards are required.

## 🔍 How to Use the Desktop App

The desktop app serves as your main dashboard. Once you open the program, you will see a simple window showing your account activity. 

- **Dashboard**: This screen shows your total token count for the month.
- **Cost Tracker**: See how much you spend based on current usage rates.
- **Alerts**: Enable notifications to get a message when you reach a set limit.

The interface updates automatically as you work. You do not need to refresh the window.

## 🖥️ Using the Terminal Interface

Cluse includes a terminal component for users who prefer working in a command window. To open the interface, press the Windows key, type "cmd," and press enter. Type `cluse` in the black window and press Enter.

The text interface provides a summary of your active projects. It shows:
- Active model versions. 
- Input and output token counts.
- Current billing session duration.

You can hide or show this window while you work on other tasks.

## 🛠️ Configuration Options

The settings menu allows you to change how the app tracks your data. You can choose different units of measurement for your tokens. If you manage multiple projects, use the profile feature to group your costs separately. This helps you understand which projects consume the most resources.

## ❓ Frequently Asked Questions

**Does the app collect my personal data?**
The app only tracks token usage numbers. It does not read your project files or store your private conversations with the AI.

**Can I run Cluse alongside Claude Code?**
Yes. The app monitors the background activity of your terminal sessions. It will detect your usage automatically once the app reaches an active state.

**Is there a cost to use Cluse?**
Cluse is free software. You are only responsible for the usage fees that Anthropic charges for the tokens you consume.

**What if the usage numbers look wrong?**
Restart the application to refresh the data connection. If the issue persists, check your internet connection.

## 📈 Tips for Better Tracking

- Keep the desktop app open while you code to monitor costs in real-time.
- Set a monthly budget limit in the settings to avoid unexpected charges.
- Check the history log once a week to review your efficiency.
- Close the terminal window when you finish your coding session to save system resources.

## 🚀 Troubleshooting

If the program fails to find your data, perform these steps:

1. Close the app completely.
2. Sign out of your terminal session.
3. Open the app again.
4. Verify your internet connection.

If you still see errors, check that your firewall allows the application to send and receive data. Cluse requires a connection to the network to sync with your usage logs.
